Project Overview

Pingoo is a comprehensive messaging application that combines the best features from WhatsApp, Telegram, and Facebook Messenger. This portfolio project showcases modern full-stack development with real-time capabilities, security features, and scalable architecture.

Why "Pingoo"?

The name combines "Ping" (message notification sound) with a friendly, modern suffix "-oo" for a unique brand identity that's easy to remember and sounds approachable.

Project Goals

  • Create a hybrid messaging app combining features from three major platforms
  • Demonstrate modern full-stack development skills
  • Implement real-time communication with WebSockets
  • Ensure security with end-to-end encryption
  • Design for scalability and performance

Platform Features

WhatsApp Features

  • End-to-end encryption for private chats
  • Status updates (24-hour disappearing stories)
  • Group chats with admin controls
  • Media sharing (images, videos, documents)
  • Voice messages with waveform
  • Read receipts and typing indicators
  • Contact synchronization

Telegram Features

  • Channels and broadcast messages
  • Bots and automation platform
  • Secret chats with self-destruct timer
  • Large file sharing (up to 2GB)
  • Username-based contacts (no phone required)
  • Edit/delete messages for everyone
  • Message pinning

Messenger Features

  • Reactions to messages (like, love, laugh, etc.)
  • Stories with filters and effects
  • Video/voice calls with WebRTC
  • Chat colors and themes
  • Polls in chats
  • Games and mini-apps integration
  • Marketplace integration (future)

Broadcast channel information

Key Relationships

  • One User can have many Contacts
  • One Chat can have many Messages
  • One Message can have many Reactions
  • One User can create many Status updates
  • Groups and Channels are specialized Chat types

MikroORM Setup with Seed Commands

$ npm run migration:create Create migration from entity changes
$ npm run migration:up Apply migrations
$ npm run seed:run Run all seeders
$ npm run db:reset Reset database with fresh data
