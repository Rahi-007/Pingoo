System Architecture

Design overview of Pingoo messaging application combining features from WhatsApp, Telegram, and Facebook Messenger

Overview

Pingoo is a hybrid messaging application combining features from WhatsApp, Telegram, and Facebook Messenger for portfolio demonstration.

Tech Stack

Backend

NestJS with TypeScript

Frontend

Next.js 14+ with TypeScript

Database

PostgreSQL 14+ with MikroORM

Real-time

Socket.io for WebSocket connections

Authentication

JWT with refresh tokens

File Storage

Cloudinary/AWS S3 for media files

Caching

Redis for session management

Encryption

Signal Protocol for E2E encryption

Core Features from Each Platform

WhatsApp Features

  • End-to-end encryption for private chats
  • Status updates (24-hour disappearing)
  • Group chats with admin controls
  • Media sharing (images, videos, documents)
  • Voice messages
  • Read receipts and typing indicators

Telegram Features

  • Channels and broadcast messages
  • Bots and automation
  • Secret chats with self-destruct timer
  • Large file sharing (up to 2GB)
  • Username-based contacts (no phone number required)
  • Edit/delete messages

Messenger Features

  • Reactions to messages (like, love, etc.)
  • Stories with filters and effects
  • Video/voice calls
  • Marketplace integration (optional)
  • Games and mini-apps
  • Chat colors and themes
