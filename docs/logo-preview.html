Implementation & Usage

All logos are provided as SVG files with embedded animations. They are scalable, lightweight, and perfect for web and mobile applications.

Favicon & App Icons

Use the square icon versions (without text) for favicons (16x16, 32x32) and mobile app icons (1024x1024).

Full Logo

Use the icon + "Pingoo" text combination for website headers, marketing materials, and documentation.

Color Variations

Each logo works on light and dark backgrounds. Use the inverted versions for contrast.

Font Recommendation

Use Inter font (semi-bold) for the "Pingoo" text to maintain consistency with the modern tech aesthetic.

Pro Tips

  • Create both square icon version and full logo (icon + text) for different use cases
  • Use SVG format for all web applications to ensure crisp rendering at any size
  • For mobile apps, export PNG versions at 1x, 2x, and 3x resolutions
  • Consider creating a monochrome version for printing and grayscale contexts
